Data Structure

Python have four type data structures which can hold large numbers variables value.They are List, Tuples, Dictionary, and Set.

List

List are good for keeping track of things by their orders, especially when the order and content might change.Unlike strings, list are mutable. We can add a new element, delete an element or change an element.We can create a new list by [] and list().

              
                
                >>> empty_list = []
                >>> weekdays = ["Monday", "Tuesday", "Wednusday", "Thursday", "Friday"] 
                >>> numbers = [12,56,89.89,67,True]
                >>> list_of_list = [weekdays, numbers]

Converting other data types into list

We can convert other data type into list by list().

              
                
                >>> list("Mad")
                ['M', 'a', 'd']
                >>> list("Hello There")
                ['H', 'e', 'l', 'l', 'o', ' ', 'T', 'h', 'e', 'r', 'e']
                >>> a_tuple = ("cat", "dog", "python")
                >>> list(a_tuple)
                ['cat', 'dog', 'python'] 
                >>> birthday = "01/06/2024"
                >>> birthday.split("/")
                >>> ['01', '06', '2024']
                >>> splitme = 'a/b//c/d//e'
                >>> splitme.split("/")
                >>> ['a', 'b', '', 'c', 'd', '', 'e']
                >>> splitme.split("//")
                ['a/b', 'c/d', 'e']

Get an Item by Index

We can get total number item of a list by len().Every item has a index in a list. The first item's index is 0.The Second item's index is 1 and so on.We can also change or update an item by it's index.

              
                
                >>> weekdays = ["Monday", "Tuesday", "Wednusday", "Thursday", "Friday"] 
                >>> total_no_of_weekdays = len(weekdays)
                >>> total_no_of_weekdays
                5
                >>> weekdays[0] 
                'Monday'
                >>> weekdays[5-1] 
                'Friday'
                >>> weekdays[0] = "Sunday"
                >>> weekdays
                ["Sunday", "Tuesday", "Wednusday", "Thursday", "Friday"] 
                >>> first_fourdays = weekdays[0:4] 
                ['Sunday', 'Tuesday', 'Wednusday', 'Thursday']
                >>> weekdays[0:4:2] #Skipping 1
                ['Sunday', 'Wednusday']
                >>> weekdays[4:0:-2] # moving from fourth index to zeroth index in reverse order by skipping 1
                ['Friday', 'Wednusday']
                >>>

Tuples

we can create a tuples by (). Tuples are basically constant storage of elemen variables . The element are not changable i.e. not editable, addable, removable.Basically tuples are used to store password,otp etc.We can sort the elements of a list by sort().

Making string from list

Converting a list to a string by join().

              
                
                >>> animals = ['Cat', 'Dog', 'Rat', 'Ant']
                >>> st = ','.join(animals)
                >>> st 
                >>> 'Cat,Dog,Rat,Ant'

Dictionaries

A dictionary is similar to a list, but the orders of items does't matter, and they aren't selected by an offset such as 0 or 1.Instead, you specify a unique key to assoc iate with each value.This key is often a string, but it can actually be any of Python's immutable types: boolean, integer, float,tuple,string, and others that you will see in later chapters.Dictionaries are mutable so we can add, delete and change their key-value elements.

Creation Dictionary

We can create a new dictionary by dict() or {} with giving key and value.

              
                
                >>> empty_dict = {}
                >>> a_dict = {"name": "Ram", "Age": 35, "Nationality":"Indian", "pan": "GHKL1234"} 
                >>> a_list = [['a',98],['b',45],["c",7]]
                >>> new_dict = dict(a_list)
                >>> new_dict 
                {'a': 98, 'b': 45, 'c': 7}

Dictionary manipulation

We can add a item with key and value.We can delete an item by key with del.We can add new items from another dictionary to a dictionary by update().Delete all items by clear().We can check presence a key by in.We can get all keys by keys() and get all values by values(). 

              
                
                >>> a_dict = {"name": "Ram", "Age": 35, "Nationality":"Indian", "pan": "GHKL1234"} 
                >>> a_dict["Height"] = "5.4ft"
                >>> a_dict 
                {'name': 'Ram', 'Age': 35, 'Nationality': 'Indian', 'pan': 'GHKL1234', 'Height': '5.4ft'}
                >>> del a_dict['Age']
                >>> a_dict 
                >>> {'name': 'Ram', 'Nationality': 'Indian', 'pan': 'GHKL1234', 'Height': '5.4ft'}
                >>> other = {"Address": "5,Circular Baligunj Road,Kolkata-700089", "Bank": "UBI"}
                >>> other.clear()
                >>> other 
                >>> {}
                >>> 'pan' in a_dict 
                >>> True
                >>> list(a_dict.keys())
                >>> ['name', 'Age', 'Nationality', 'pan'] 
                >>> list(a_dict.values())
                >>> ['Ram', 'Indian', 'GHKL1234', '5.4ft']
                >>> list(a_dict.items())
                [('name', 'Ram'), ('Nationality', 'Indian'), ('pan', 'GHKL1234'), ('Height', '5.4ft')]
                >>>

Sets

A set is like a dictionary with its values thrown away.The set only have containing unique element.We can create set by set() or {}.

              
                
                >>> empty_set = set()
                >>> empty_set
                set()
                >>> even_numbers = {0,2,4,6,8,10}
                >>> a_set = set('Kalbosh')
                {'b', 'l', 'a', 's', 'o', 'K', 'h'}
